# How to start the app
Run the `main.py` file with Python 3.13 or newer.

# How to use my own data
Add your recurrence/distance matrix to the list `matrices` in the `main.py` file and specify a label and wether it is a recurrence matrix.

For the example data, we used `numpy.savetxt()` to save the recurrence/distance matrix as a text file and then import it with `numpy.loadtxt`.
